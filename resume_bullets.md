# Resume Bullets for AI Enterprise Insight Agent

- Built an AI Enterprise Insight Agent using Python, Streamlit, Groq/Llama, ChromaDB, sentence-transformer embeddings, and LLM prompting to retrieve enterprise context and generate grounded business responses.
- Implemented a Retrieval-Augmented Generation pipeline with document chunking, vector database storage, semantic search, and context-aware answer generation.
- Integrated structured CSV analysis using Pandas to summarize sales, support tickets, customer satisfaction, and regional performance metrics.
- Developed an AI action planner that converts business problems into structured recommendations using retrieved context and operational metrics.
- Applied prompt engineering and source-aware retrieval to improve answer reliability, reduce hallucination, and support enterprise AI governance.
